# MERN E-commerce App

## 📦 Tech Stack

- MongoDB + Mongoose
- Express.js
- React.js + Vite
- Node.js
- Stripe for Payments
- Tailwind CSS

---

## 🚀 Setup Instructions

### Clone the Repo




Backend Setup



cd backend
npm install
cp .env.example .env
# Fill your MongoDB URI and Stripe keys
npm run dev





Frontend Setup


cd frontend
npm install
npm run dev
